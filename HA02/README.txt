ToDo Tracker Version 1.0 11/05/2017

Beschreibung
-------------

- diese Website simuliert einen ToDo Tracker

Vorraussetzungen
-----------------
- um sie zu starten wird eine lokale Eclipse Anwendung mit dem Java EE Developers Package vorrausgesetzt, 
  diese ist falls nötig auf http://www.eclipse.org zu erhalten

- die HA02.zip Datei muss im Eclipse importiert werden

- für eine erfolgreiche Anbindung der Datenbank muss sichergestellt werden, dass der Pfad der datenbank.db in der Java Klasse todoEintragBean angepasst wird

- Die Anwendung wird mit dem Tomcat Server gestartet
